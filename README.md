# 💊 Sortret

**Android-приложение для отслеживания курса приёма препарата Сортрет**

*Ведёт дни курса · считает накопление мг/кг · рисует живое жидкое стекло*

---

> ⚕️ **Важно.** Приложение не является медицинским инструментом и не заменяет консультацию врача. Все расчёты носят ознакомительный характер.

## ✨ Возможности приложения

* **📅 Управление курсом**: Ввод веса, даты начала, общей целевой дозы и автоматический расчёт примерной даты окончания.
* **⏰ График приёма**: Настройка времени и дозировки для утреннего и вечернего приёмов, а также учёт текущего остатка таблеток.
* **📊 Статистика прогресса**:
  * Общий процент завершения курса.
  * Накопленная доза в мг и мг/кг.
  * Динамика активного вещества в организме (с учётом полувыведения ~20 ч).
  * Плавные Compose-анимации заполнения прогресса.
* **🎨 Живое жидкое стекло (Glassmorphic UI)**:
  * Анимированный Canvas-фон с цветными blob-облаками.
  * Детальная настройка стекла: скругление, высота преломления, смещение, контраст, белая точка, тонировка, размытие и хроматическая дисперсия.
  * Установка кастомного изображения на фон с автоопределением акцентного цвета.
  * Встроенные пресеты фонов («Рубиновый сон», «Синий ирис», «Космический гранат» + расширенный список).

```

│   │   ├── logic/

│   │       ├── components/
│   │       │   ├── AnimatedBackground.kt     # Canvas-рендеринг живых облаков

│   │       │   ├── GlassCard.kt              # Базовый контейнер с drawBackdrop и эффектами
│   │       │   ├── BackdropDialog.kt         # Универсальные размытые диалоги
│   │       │   └── ...                       # Вспомогательные слайдеры, поля ввода

```

## 🔬 Математика расчётов

1. **Кумулятивная доза**:
   Приложение итерирует по дням с даты начала до текущего момента. Учитывается время приёмов и выбранный первый приём в первый день курса (если выбран вечерний, утренняя доза первого дня пропускается).
2. **Активное вещество (распад)**:
   Вычисляется экспоненциальный распад доз:
   $$C_{\text{active}} = \sum \text{Dose}_i \cdot e^{-\lambda \cdot t_i}$$
   где $\lambda = \frac{\ln(2)}{T_{1/2}}$, а $T_{1/2} = 20\text{ часов}$.
3. **Дата окончания**: 
   $$\text{Дни курса} = \lceil \text{Целевая доза} / \text{Суточная доза} \rceil$$
   Окончание рассчитывается прибавлением дней к дате начала.

## 🚀 Сборка и установка

### Сборка через скрипт (Windows)
Запустите `build-apk.bat`. Скрипт автоматически:
1. Найдёт JDK в стандартной директории Android Studio.
2. Соберёт отладочный APK (`sortret-v2.8.apk`).
3. Установит его на подключённое устройство через ADB с **сохранением пользовательских данных** (`adb install -r`).

### Сборка через Android Studio
1. Откройте директорию `sortret` в Android Studio.
2. Проверьте путь к SDK в `local.properties`.
3. Подключите устройство и запустите конфигурацию `app`.

## ⚠️ Ограничения

* **Android 13+**: Эффект преломления линзы (shader) работает только на Android 13 и новее. На более старых версиях преломление автоматически отключается.
* **Производительность**: На слабых устройствах Canvas-анимация фона совместно с Backdrop-размытием может вызывать повышенный расход батареи.
