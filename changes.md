# Список изменений в версии 2.9 (Release Notes)

1. **🔗 Двусторонний связанный ввод целевой дозы:**
   * В окне настроек курса вместо одного поля ввода дозы в мг теперь находится строка с двумя полями:
     * **Слева:** Целевая доза в **мг** (общая цель курса).
     * **Справа:** Целевая доза в **мг/кг** (вычисляемая накопительная цель по отношению к весу, например, 120 или 150 мг/кг).
   * Поля связаны математически: при редактировании дозы в мг автоматически обновляется показатель мг/кг, и наоборот.

2. **🔴 Кнопка полного сброса прогресса:**
   * В окно настроек (раздел «Дополнительно» в визуальных параметрах) добавлена кнопка **«Сбросить весь прогресс»**.
   * Кнопка выделена красным цветом для обозначения деструктивного действия.
   * Нажатие на неё полностью обнуляет весь прогресс курса: очищает календарь ручных отметок, сбрасывает запасы таблеток, утерянные капсулы, вес, целевые дозы и график приёма к начальным установкам.

3. **🎨 Редизайн README.md и добавление скриншотов:**
   * Описание проекта полностью переписано живым языком разработчика (без канцелярских фраз ИИ).
   * Добавлено красивое адаптивное отображение скриншотов из папки `pics` в ряд (сетка 3x1).
   * Заменена нечитаемая формула LaTeX на аккуратный текстовый код, удалена громоздкая история версий.

4. **🚀 Версия 2.9 и компиляция:**
   * Версия приложения обновлена до `2.9` (code `29`).
   * Собраны бинарные файлы:
     * `Sortret_v2.9.apk` (подписанный дебаг-билд для установки на телефон).
     * `Sortret_v2.9_Release_Unsigned.apk` (оптимизированный R8-сжатый релизный билд без подписи).
   * Выполнен деплой на подключенное устройство через ADB с сохранением пользовательских данных.
   * Вся папка проекта очищена от мусора сборки и временных кэшей.
